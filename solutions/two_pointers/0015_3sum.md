# 15. 3Sum

| Field | Value |
|---|---|
| **Difficulty** | Medium |
| **Pattern** | Two Pointers |
| **Time Complexity** | O(n^2) |
| **Space Complexity** | O(1) |
| **Language** | python3 |
| **Solved On** | 2026-03-14 |
| **Tags** | Array, Two Pointers, Sorting |

## Key Insight

The solution leverages sorting and the two-pointer technique to efficiently find triplets that sum to zero, avoiding duplicates by skipping over repeated elements.

## Review Tip

## Solution

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        result = []
        nums.sort()

        for i, a in enumerate(nums):
            if i > 0 and nums[i-1] == a:
                continue
            
            l, r = i + 1, len(nums) - 1

            while l < r:
                threesums = a + nums[l] + nums[r]
                if threesums > 0:
                    r-=1
                elif threesums < 0:
                    l+=1
                else:
                    result.append([a, nums[l], nums[r]])
                    l+=1
                    while nums[l] == nums[l-1] and l < r:
                        l+=1

        return result
        
```
